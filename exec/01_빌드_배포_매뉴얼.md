## 4) 빌드/실행 절차

### A. 백엔드
```bash
cd be/S14P11A201/zerojuice
./gradlew bootRun --args='--spring.profiles.active=local'
# 또는
./gradlew build
java -jar build/libs/*.jar
```
- Swagger: `http://localhost:8081/api/v1/swagger-ui.html` (local 프로필 기준)

### B. 관리자 웹
```bash
cd fe/S14P11A201/my-react-app
npm install
npm run dev
# 배포 빌드
npm run build
```
- Docker 배포 시 `nginx`로 정적 파일 서빙 (`80` 포트)

### C. 모바일 앱
```bash
cd app/S14P11A201/smart-parking
npm install
npm run start
npm run android
```

### D. AI
```bash
cd ai/S14P11A201
pip install opencv-python numpy python-socketio requests python-dotenv urllib3
python camera_test.py
python roi_editor.py
python parking_slot_detection.py
```

### E. 임베디드(예시)
```bash
# RF2O
ros2 launch rc_nav2_bringup rf2o.launch.py use_sim_time:=False scan_topic:=/scan rf2o_freq:=5.0

# SLAM+내비
ros2 launch rc_nav2_bringup bringup_all.launch.py

# 저장된 맵 기반 주행
ros2 launch rc_nav2_bringup navigation.launch.py
```
